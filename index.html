Formula Reference

Every governing equation used across the three calculators, with variable definitions, units, assumptions and practical ANSYS Fluent notes.

01Isentropic flow relations

Recovers static temperature and pressure from stagnation conditions and Mach number, then density and speed of sound follow from the ideal gas law.

T = T₀ / (1 + 0.5(γ−1)M²)
P = P₀ / (1 + 0.5(γ−1)M²)^(γ/(γ−1))
ρ = P / (R·T)
a = √(γ·R·T)
U = M·a

02Sutherland's viscosity law

Gives dynamic viscosity as a function of static temperature, matched to the reference values Fluent uses by default for air.

03Wall units & first cell height

Rather than estimate skin friction analytically, the Bias Factor calculator reverse-engineers it from a converged Fluent wall-shear-stress report, which is more accurate for the actual geometry than a flat-plate correlation.

C_f = τ_w / (0.5·ρ·U²)
U_τ = √(τ_w / ρ)
Δy = y⁺ · μ / (ρ·U_τ)

04Geometric bias series (mesh growth)

A one-sided biased row of N cells with first height Δy and constant growth ratio r forms a finite geometric series that must sum to the domain length L. The symmetric case mirrors the series from both ends toward a shared center.

One-sided: L = Δy · (1 − r^N) / (1 − r)
Symmetric, N even: L = 2·Δy · (1 − r^(N/2)) / (1 − r)
Symmetric, N odd: L = 2·Δy · (1 − r^⌊N/2⌋) / (1 − r) + Δy·r^⌊N/2⌋

Solved for r via bisection (bracketing search, tolerance 1×10⁻¹², up to 1000 iterations) when N is known; solved algebraically for N when r is known, then re-solved for the exact r that fits the integer division count.

N (from target r): N = ln(1 − L(1−r)/Δy) / ln(r)

05CFL condition & time step

The Courant–Friedrichs–Lewy number relates a time step to how far information can travel across one cell in that step. Given a target CFL and the smallest cell in the mesh, the required time step follows directly.

Δt = CFL · Δx / U
t_residence = L / U
t_total = t_residence · N_residence
N_steps = ⌈t_total / Δt⌉

06Flat-plate turbulent boundary layer & inflation stack

Used only by the Inflation Layer calculator, where no Fluent report yet exists to read τ_w from. These are the standard Schlichting flat-plate correlations, valid roughly over 5×10⁵ < Re_x < 1×10⁷, and are the same relations behind ANSYS Fluent Meshing's built-in Y+ estimator.

Re_x = ρ·U·x / μ
C_f = 0.058 · Re_x^(−0.2)
δ = 0.37 · x / Re_x^(0.2)
τ_w = 0.5·ρ·U²·C_f   U_τ = √(τ_w/ρ)

The layer stack itself is a finite geometric series exactly as in the bias calculator, letting you solve total thickness from a layer count, or the layer count needed to reach a target percentage of the estimated boundary-layer thickness δ:

Total thickness = y₁ · (r^N − 1) / (r − 1)
N (for target thickness) = ln(1 + total·(r−1)/y₁) / ln(r)

About this tool

Built for the pre-processing bench

Y+ Console exists to remove one specific class of mistake: hand-typing mesh-sizing numbers into a spreadsheet, or a Fluent journal, and getting a decimal place wrong at 1am before a submission deadline.

What it is

A set of three focused calculators — Bias Factor, CFL & Time Step, and Inflation Layers — covering the sizing decisions made before and after a first CFD pass. The bias and CFL calculators are direct ports of validated reference scripts; the inflation calculator adds standard flat-plate correlations for cases with no prior solution to read shear stress from.

What it isn't

Not a solver, not a mesh generator, and not a substitute for judgment on your specific geometry. Flat-plate correlations are a starting estimate for separated, curved, or highly three-dimensional flows — validate against your own y+ report once a solution exists.

How it computes

Every calculation runs in your browser in plain JavaScript — no server round-trip, no stored inputs. The bias and CFL logic is a line-by-line translation of the supplied Python reference implementations, including the bisection solver's exact tolerance and iteration cap.

Who it's for

Aerospace and mechanical engineering students and researchers running compressible CFD cases in ANSYS Fluent or OpenFOAM, who want mesh-sizing numbers they can trust without re-deriving them by hand each time.
